📋 How to Test

  1. Load the Extension: Make sure the Annotate-Translate extension is installed and enabled
  2. Select Text: Click on any test text below and select it

✅ Verification Checklist

After testing, verify that:

  • ✓ <b> and <strong> tags render as bold blue text
  • ✓ <i> and <em> tags render as italic red text
  • ✓ <u> tags render with blue underline
  • ✓ <mark> tags render with yellow highlight
  • ✓ No <script> or dangerous tags are executed
  • ✓ Plain text without tags still works normally
  • ✓ Both translation popup and annotation detail popup show formatted examples

🔒 Security

The sanitizeHTML() function ensures that only safe formatting tags are allowed:

  • ✅ Allowed: <b>, <i>, <em>, <strong>, <u>, <mark>, <span>
  • ❌ Blocked: <script>, <iframe>, <img>, event handlers, etc.
  • 🛡️ All HTML is first escaped, then only safe tags are restored
