Phonetic & Pronunciation Settings
When translation provider doesn't return phonetic information, automatically fetch it from external sources. Currently supports English phonetics (via FreeDictionary API). Future support planned for Chinese Pinyin and other languages.
💡 How it works:
1. Translation provider processes the word → Returns translation and definitions
2. If no phonetics returned → Automatically queries external API → Gets phonetics
3. Result: Complete information with translation + definitions + phonetics

✅ Current support: English phonetics (single words, via FreeDictionary API - free, no registration)
