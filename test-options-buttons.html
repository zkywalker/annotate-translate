技术实现

修改的代码：

<!-- 之前 --> <button class="btn btn-primary" id="saveButton"> 💾 <span data-i18n="saveSettings">Save Settings</span> </button> <!-- 现在 --> <button class="btn btn-primary" id="saveButton"> <i data-lucide="save" width="18" height="18"></i> <span data-i18n="saveSettings">Save Settings</span> </button>

新增的样式：

.btn { display: flex; align-items: center; justify-content: center; gap: 8px; } .btn i[data-lucide] { flex-shrink: 0; }

优势：

  • 图标和文字使用 flexbox 完美对齐
  • 8px 的间距（gap）让布局更舒适
  • 图标不会被压缩（flex-shrink: 0）
  • 跨浏览器、跨平台显示一致
  • 易于维护和扩展
