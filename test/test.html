🔖 Annotate Translate Extension - Test Page

📋 测试说明：

  1. 打开 Chrome DevTools（按 F12）
  2. 切换到 Console 标签查看调试日志
  3. 悬浮窗方式：选中文字后会出现小巧的悬浮窗，显示 T（翻译）和 A（标注）按钮
  4. 右键菜单方式：右键点击选中的文字，选择 Annotate "..."
  5. 在弹出的对话框中输入标注（如拼音、翻译等）
  6. 观察文字上方是否出现标注

💡 提示：悬浮窗现在更小巧，不会影响阅读！按钮只显示单字母，悬停时会显示完整说明。
