代码示例

HTML 中使用

<i data-lucide="settings" width="20" height="20"></i> <i data-lucide="volume-2" width="16" height="16"></i>

JavaScript 中创建

const icon = document.createElement('i'); icon.setAttribute('data-lucide', 'volume-2'); icon.setAttribute('width', '16'); icon.setAttribute('height', '16'); // 初始化图标 lucide.createIcons();

样式定制

.audio-button { background: transparent; color: #5f6368; } .audio-button:hover { background: #e8f0fe; color: #1a73e8; }
