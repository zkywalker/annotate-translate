# Claude Code Global Guidelines

This file contains global guidelines for Claude Code when working in this repository.

## Git Commit Guidelines

When creating git commits:
- Write commit messages in **English only**
- Keep the summary concise and factual
- No excessive formatting or decorative elements
- Follow conventional commit format: `feat:`, `fix:`, `refactor:`, etc.
- Example: `feat: add abort controller for vocabulary mode translation`
