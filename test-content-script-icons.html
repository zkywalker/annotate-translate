🔍 内容脚本图标加载测试

1. 检查 Lucide 库加载状态

点击按钮检查 Lucide 库是否加载...

2. 测试动态创建图标

点击按钮动态创建图标元素...

3. 测试手动初始化

点击按钮手动调用 lucide.createIcons()...

4. 测试翻译功能

选中这段文字进行翻译测试：
Hello world! This is a test for translation feature. The audio button should display the Lucide volume-2 icon.
选中上方文字，等待翻译弹窗出现，检查音频按钮图标是否显示。

5. 控制台日志检查

打开开发者工具控制台（F12），应该看到：
  • [Lucide] Loaded successfully from extension
  • [Lucide] Icons initialized
如果看到这些日志，说明 Lucide 加载成功。

6. 预期结果

✅ 成功标志：
  • 动态创建的图标应该立即显示为 🔊 图标（volume-2）
  • 翻译弹窗的音频按钮应该显示图标
  • 标注中的音频按钮应该显示图标
  • 所有图标都可以正常点击
  • 控制台无 CSP 错误
